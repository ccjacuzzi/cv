HTML Input Type Attribute Options

For a detailed explanation and a complete list of examples, visit mdn web docs_: input types.

Examples

Type Description Example
Button A push button with no default behavior. Don't use. Use the the button element instead.
Checkbox A check box allowing single values to be selected/deselected.
Color A control for specifying a color. Opens a color picker in supported browsers.
Date A control fro entering a date.
File A control that lets a user select a file.
